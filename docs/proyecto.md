# Proyecto: Servicio de predicción (Titanic)

## Objetivo

Desarrollar un servicio sencillo de predicción basado en un modelo de Machine Learning.
El modelo predice la supervivencia de pasajeros del Titanic a partir de un dataset público.

El foco principal de este entregable es demostrar el uso correcto de Git y GitHub:
organización, equipos, ramas, commits, Pull Requests, Issues, Projects, CODEOWNERS
y políticas de protección de ramas.
